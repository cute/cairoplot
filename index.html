CairoPlot was born as an attempt to plot beautiful charts for presentations, papers and websites. Using python and pycairo, it's possible to create 8 different cool kinds of charts.

As of this version, it's possible to create 8 different types of charts: horizontal and vertical bars, scatter, dot lines and function plots, pie, donut and gantt charts.

This project is maintained by Rodrigo Araujo with contributions from Mike Rooney, Joao Bueno, Magnun Leno, Paul Hummer, Karel Kolman and many others
